## Installation

### Pre-requisites

#### Mac OSX

```sh
brew install pango
```

#### Linux (debian/ubuntu)

```sh
sudo apt-get install libpango-1.0-0 libpangoft2-1.0-0
```

### Pebblo Daemon

```sh
pip install pebblo
```

## Run Pebblo daemon

```sh
pebblo
```
